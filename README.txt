# Joby Challenge: networking
This is library intended to ping addresses and find addresses that share host
address (last octet) but have different network adresses (first three octets).

# Installation
- Clone via GitHub
- Open command line in aeropy directory
- Run 'pip install -e .'
- cd tests
- python -m unittest
- If all tests pass, the library is properly installed

# Dependencies
There are no dependencies.
